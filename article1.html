How to Build a Website from Scratch in 3 Days

Published on March 20, 2025 • Tutorial

Introduction

Building a website from scratch might seem daunting, but with the right plan, you can create a simple, functional site in just 3 days! This guide is for beginners who know the basics of HTML, CSS, and JavaScript. By the end, you'll have a personal portfolio site that you can share with the world.

What you'll need:

  • A code editor (like VS Code)
  • A browser (like Chrome)
  • 3 days of focused work (2-3 hours per day)

Day 1: Planning and Basic Structure (HTML)

Step 1: Choose a Theme

Let's build a personal portfolio site to showcase your skills. It will have a homepage with a header, an "About Me" section, a "Projects" section, and a contact form.

Step 2: Set Up Your Project

Create a new folder called portfolio. Inside it, create three files: index.html, styles.css, and script.js.

Step 3: Create the HTML Structure

Open index.html and add the following HTML structure:

Day 3: Adding Interactivity and Publishing

Step 1: Add a "Back to Top" Button

The "Back to Top" button is already in your index.html. Now, let's add the JavaScript to make it work. Open script.js and add the following code:

window.onscroll = function() {
    const backToTopButton = document.querySelector('.back-to-top');
    backToTopButton.style.display = window.scrollY > 300 ? 'block' : 'none';
};

function scrollToTop() {
    window.scrollTo({ top: 0, behavior: 'smooth' });
}

Step 2: Publish Your Site

You can host your site for free using GitHub Pages:

  1. Create a GitHub account if you don't have one.
  2. Create a new repository named [your-username].github.io.
  3. Upload your portfolio folder (containing index.html, styles.css, and script.js) to the repository.
  4. Go to the repository settings, scroll to "GitHub Pages," and set the source to the main branch.
  5. Visit https://[your-username].github.io to see your live site!

Conclusion

Congratulations! You've built a website from scratch in just 3 days. You now have a personal portfolio with a clean design, a contact form, and a "Back to Top" button. To take it further, consider adding more projects, improving the design, or learning a framework like React.
